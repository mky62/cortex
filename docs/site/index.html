Cortex Documentation

Documentation for the Cortex AI customer support platform — a monorepo building an AI-powered customer support system with chat and voice capabilities.

Quick Start

# Install dependencies
pnpm install

# Start development
pnpm dev

# Web dashboard: http://localhost:3000
# Widget: http://localhost:3001?organizationId=<YOUR_ORG_ID>
